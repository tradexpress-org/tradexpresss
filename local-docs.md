# Local Documentation

## Local Library
A local library is a structured collection of records, articles, and references stored in the workspace for offline use and quick browsing.

## Local Type
Local type refers to the local classification and tagging of each record, such as article, media, directory, or reserved node.

## Local Web
Local web is the local interface and navigation layer that makes these records accessible through the app UI, links, and grouped directories.

## Autoflow
Autoflow is the automatic chain of actions that moves a local record from capture to organization, including search, grouping, linking, and media recording.

## Current Usage
- Library entries are sourced from articles.json.
- Records can be searched, filtered, and grouped into directories.
- Quick links provide access to Contact, Account, Passcode, and Webauth.
- Media and reserve sections are available for future expansion.
